# ThePerksofUsingTunepad
This project was undertaken as a Working with Data and Code (52685) Assessment 2 task.


## General Information
- This is a composition created using TunePad, meant to accompany the tunnel scene in The Perks of Being a Wallflower (2012) film.
- This project was initially intended to determine how music changes the emotive response to film scenes.


## Technologies Used
- TunePad - coded using Python.


## Usage
To view this project, use this [link](https://tunepad.com/project/114919) to open it in TunePad. 

When playing the composition in TunePad, do not press the play button in the top right corner when on the 'code' page. In order for the piece to play without technical difficulties, the composition must be played from the 'tracks' page.

Note, when playing the audio on TunePad, the audio will stop playing if you enter a different window or tab.

View the completed composition accompanying the film scene [here](https://youtu.be/C9XHCZu5EOA).
